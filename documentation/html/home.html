What is Angular Validate This Form?

Validate This Form is a helper directive that adds ng-messages validation messages to your form dynamically and styles your Bootstrap form automatically for you.

View GitHub Repository View Documentations

Installation & Usage

  • Install it using bower:

    bower install angular-validate-this-form

  • Reference it in your Angular app:

    angular.module('yourApp', [ 'angular-validate-this-form' ]);

  • Apply the directive on your form:

    <form class="form-horizontal no-margin" name="myForm" novalidate validate-this-form>
    </form>
